## Azure

1. Navigate to Microsoft Registered App section. Under Microsoft Entra ID, open App Registration.

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240430-103108.png" alt="Navigate to Microsoft Registered App" style={{ width: '40%', height: 'auto' }} />

2. Select new registration.

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240430-103521.png" alt="Select New Registration" style={{ width: '40%', height: 'auto' }} />

3. The Created Application will contain client id, client secret which will be used in following steps.

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240430-103836.png" alt="Client ID and Secret" style={{ width: '40%', height: 'auto' }} />

4. Under optional claims, add the necessary fields like email. (for a field to work, azure users should have the respective details present in their account)

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240430-104251.png" alt="Add Optional Claims" style={{ width: '40%', height: 'auto' }} />

5. Go to Authentication and fill in the redirect URL.

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240430-104922.png" alt="Fill in Redirect URL" style={{ width: '60%', height: 'auto' }} />
