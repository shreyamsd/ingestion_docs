To use Google SSO to login to the Vue Platform, you would need to do the following:

1. In your Google Cloud Console, navigate to the APIs & Services section within your Google Cloud Service.

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240527-142050.png" alt="Navigate to APIs & Services" style={{ width: '30%', height: 'auto' }} />

2. Next, select Credentials.

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240422-100644.png" alt="Select Credentials" style={{ width: '30%', height: 'auto' }} />

3. Create an OAuth Client ID credentials and choose Application type as Web application.

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240422-100901.png" alt="Create OAuth Client ID" style={{ width: '40%', height: 'auto' }} />

4. Send the Client ID, Client Secret from the created Application to us at Vue.

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240527-142217.png" alt="Copy Client ID and Secret" style={{ width: '40%', height: 'auto' }} />

5. We will generate a Redirect URI which needs to be added to Authorized redirect URI's in your application.

<img src="https://d1r1e7xjkfj7nz.cloudfront.net/image-20240527-142814.png" alt="Copy & Paste Redirect URIs" style={{ width: '60%', height: 'auto' }} />
